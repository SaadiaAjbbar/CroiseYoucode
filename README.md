# WorkSphere – Gestion du personnel

## Description du projet
ce projet est une plateforme interne de l'entreprise WorkSphere  qui permet de gérer l’affectation des employés dans différentes zones de travail.

## Objectifs
- Afficher les zones de travail de l’entreprise.
- Afficher la liste des employés et leurs rôles.
- Ajouter un employé compatible dans une zone.
- Mettre à jour sa position automatiquement.

## Fonctionnalités principales

### 1. Affichage des zones
Chaque zone contient :
- Un bouton pour afficher les employés disponibles.
- Une liste des employés affectés.

### 2. Sélection des employés
Lors du clic sur une zone :
- Affichage des employés compatibles.
- Possibilité de les ajouter dans la zone.

### 3. Mise à jour dynamique
- L’employé est retiré de son ancienne zone.
- La nouvelle zone s’actualise automatiquement.
